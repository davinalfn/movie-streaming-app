Website streaming film dengan fitur autentikasi dan CRUD menggunakan Laravel (Backend) dan Vue.js (Frontend).
Fitur

-  Register & Login User (Laravel Sanctum)
-  CRUD Movies (Create, Read, Update, Delete)
-  Search Movies
-  Video Player

 Backend:
- Laravel 10/11
- MySQL
- Laravel Sanctum

Frontend:
- Vue.js 3
- Pure CSS
